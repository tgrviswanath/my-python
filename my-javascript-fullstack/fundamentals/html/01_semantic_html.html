What is semantic HTML?

Semantic HTML uses elements that describe their meaning to both the browser and the developer. Examples include <article>, <nav>, <header>.

Why does it matter for SEO?

Search engines use semantic structure to understand content hierarchy and relevance, improving indexing and ranking.
