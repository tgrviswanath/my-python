## 🎯 Interview Tips

1. **JS internals first** — event loop, closures, prototypes are asked everywhere
2. **React hooks deeply** — useState, useEffect, useCallback, useMemo, useRef
3. **Async patterns** — Promise chaining, async/await, error handling
4. **System design** — always discuss trade-offs, not just solutions
5. **Code quality** — clean, readable, with error handling
